Oque é git, github e modularização?

Git é um sistema para controle de versões utilizado principalmente para desenvolvimento de softwares

Github é uma plataforma que utiliza do sistema git, onde usuários cadastrados públicam codigos fonte e podem trabalhar até mesmo em equipes editando o repositório, é basicamente um rede social de desenvolvedores

modularização é dividir um projeto em diversas partes para que facilite a manutenção ou modificação do mesmo


Versão do sistema é o mesmo que versionar o codigo?

não, versionar um codigo é a quando um projeto sofre alterações e a cada alterações ele recebe uma nova versão
